## Uso offline (descarga local)

Si quieres usarla sin internet:

1. Ve al repositorio en GitHub y pulsa **Code → Download ZIP**
2. Descomprime el ZIP en cualquier carpeta de tu ordenador
3. Abre el archivo `index.html` con tu navegador (doble clic)

No necesitas servidor ni nada más. Todo funciona en local.

## Cómo funciona la app

- Selecciona una asignatura desde la pantalla de inicio
- Elige entre **examen completo** (todas las preguntas en orden aleatorio), **por tema** o **repasar errores**
- Al responder, se muestra si es correcta o incorrecta y una breve explicación
- La app guarda tu progreso en el navegador (localStorage); si borras el historial del navegador, el progreso se reinicia
- Compatible con móvil, tablet y escritorio
